Linux是一个操作系统，可以用来管理计算机

Linux是一个多用户的操作系统

Linux系统中为每个使用者起了一个名字，这个名字
    叫做这个使用者的帐号。不同使用者的帐号不同。

终端是使用Linux系统的主要方式

终端里只能使用键盘不能使用鼠标

在终端窗口里通过输入不同的命令来使用计算机

whoami命令可以用来察看当前使用的帐号

clear命令可以用来清除终端窗口里的文字信息

操作系统中用来管理文件的功能叫文件系统
文件系统采用分组方式管理所有的文件
每个分组中还可以再次进行分组
每个分组叫做一个文件夹或目录
文件夹中可以包含文件和其他文件夹

如果文件夹a直接包含文件夹b则它们之间存在父子关系
    其中文件夹a叫做父目录，文件夹b叫做子目录

Linux的文件系统中有一个大分组，其中包含了
    所有文件。
这个分组叫做根目录，可以用/表示根目录

在文件系统中使用路径表示文件或文件夹的位置
绝对路径的起点是根目录(/)
所有绝对路径一定以/做开头

相对路径可以把任何一个目录作为起点
相对路径中使用..表示向左走一步(父目录)
相对路径中使用.表示原地踏步
相对路径中不需要指定起点位置

终端窗口中可以设置所有相对路径的起点，此后的相对
    路径都以它为起点
这个位置叫做终端窗口的当前目录
随时可以使用命令修改当前目录的位置

pwd命令可以用来察看当前目录的位置

Linux系统为每个使用者保留了一个私有目录，这个
    目录叫做这个使用者的HOME目录。
我们可以在终端窗口中使用~表示当前使用者的HOME
    目录
每次打开终端窗口的时候当前目录就是当前帐号的
    HOME目录

cd命令可以用来修改当前目录的位置，使用方法如下
cd 目录路径
命令执行后路径对应目录成为当前目录

ls命令可以察看一个目录中的内容,使用方法如下
ls 目录路径
如果要察看当前目录中的内容可以省略路径
-a选项可以察看目录中所有内容
-l选项可以察看每个内容的详细信息
-a选项可以和-l选项合并成-al

touch命令可以用来创建新的文件,使用方法如下
touch 文件路径
命令执行后路径对应文件出现
如果路径对应文件已经存在则它的最后被修改时间
    改成执行touch命令的时间

rm命令可以用来删除一个已经存在的文件,使用方法如下
rm 文件路径
命令执行后路径对应文件被删除

mkdir命令可以用来创建新的文件夹，使用方法如下
mkdir 目录路径
命令执行后路径对应目录被创建
mkdir命令要求路径对应的目录还不存在，但是它的
    父目录应该已经存在了
使用-p选项可以把路径中还不存在的所有目录都创建
    出来

rmdir命令可以用来删除一个目录，使用方法如下
rmdir 目录路径
命令执行后路径对应目录被删除
这个命令只能用来删除没有内容的目录

rm命令加-rf选项可以删除任何目录（不论里面是否
    有内容）

在命令中只需要输入路径中前几个字母（这几个字母
    足以唯一确定一个路径）然后按tab键计算机会
    自动把路径中其余的字母补齐

可以在终端窗口中使用上下方向键找到以前执行过的
    任何一个命令，修改后作为新命令使用

tar命令可以用来把一组文件或文件夹打包成一个
    单一文件
在打包完成后可以顺便对得到的结果文件进行压缩
    处理
打包压缩命令如下
tar zcvf 压缩文件名.tar.gz 路径1 路径2 ...
这个命令可以把所有路径对应的文件夹或文件
    打包压缩成一个文件

解压缩解包命令如下
tar zxvf 压缩文件名.tar.gz

vi是Linux系统中用来记录文字信息的工具

vi只能在终端窗口里使用，所以也不能使用鼠标
vi的功能被分成三组，任何时候只能使用其中一组
    功能
vi中包含三种工作模式，每种工作模式对应一组
    功能。
在vi中一定处于某种工作模式下

三种工作模式分别是正常模式，插入模式和命令模式
正常模式下只能执行一些简单命令
插入模式下可以修改文字内容
命令模式下可以执行复杂命令

每次进入vi的时候一定处于正常模式

不同工作模式之间可以互相转换
在正常模式下使用i可以进入插入模式
在正常模式下使用:可以进入命令模式

任何时候使用esc可以进入正常模式

可以使用如下命令启动vi
vi 文件路径
vi启动后会在终端窗口中显示文件内容

可以采用如下两种方法退出vi
1.在命令模式下输入q!可以退出vi（这种退出方式会
    丢弃所有没保存的修改）
2.在命令模式下输入wq或者x退出vi（这种退出方式会
    首先保存所有修改然后再退出）

在正常模式下输入nyy命令可以把当前光标所在行
    开始的连续n行拷贝到剪贴板上。（n代表一个
    整数，如果n是1可以省略不写）
在正常模式下输入np命令可以把剪贴板上的内容
    在当前光标所在行的下面粘贴n次（n代表
    一个整数，如果n是1可以省略不写）
在正常模式下输入ndd命令可以把当前光标所在行
    开始的连续n行剪切到剪贴板上（n代表一个
    整数，如果n是1可以省略不写）

在命令模式下使用set命令可以让vi提供各种高级
    功能
set nu命令可以让vi为每行显示一个行号

~/.vimrc文件中可以记录大量set命令，每次vi启动
    后都会从这个文件中得到这些set命令。把他们
    按顺序执行后才让用户使用。
